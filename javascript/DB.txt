    ALTER (테이블 정보 수정)

    컬럼추가
    ALTER 테이블명 ADD COLUMN 컬럼명 자료형 제약조건;

    컬럼변경
    ALTER 테이블명 MODIFY COLUMN 컬럼명 자료형 제약조건;

    컬럼변경(이름까지)
    ALTER 테이블명 CHANGE COLUMN 컬럼명 바꿀컬럼명 제약조건;

    컬럼삭제
    ALTER 테이블명 DROP COLUMN 컬럼명;

    테이블 이름변경
    ALTER 테이블명 RENAME 바꿀테이블명;

    조인(JOIN)

    SELECT와 더불어 가장 많이 사용
    두 개 이상의 테이블을 묶어서 하나의 결과로 만들어내는 명령어

    - * INNER JOIN
            양쪽 테이블 모두 있어야만 검색 결과 확인 가능

    ex) 
        SELECT 컬럼명1, 컬럼명1 || * FROM 테이블명 
            INNER JOIN 참조할 테이블명
            ON 검색조건
        WHERE 조건;

        축약
            SELECT * FROM 테이블명1, 테이블명2 WHERE 테이블1 컬럼명 = 테이블 2 컬럼명

    - OUTER JOIN
        한쪽에만 있어도 검색 결과 확인
        
    ex)
        SELECT 컬럼명1, 컬럼명1 || * FROM 테이블명 
            LEFT | RIGHT | FULL OUTER JOIN 참조할 테이블명
            ON 검색조건
        WHERE 조건;
